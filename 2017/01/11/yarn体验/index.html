yarn体验

安装

如果已经安装了npm,可以用npm install yarn -g全局安装,也可以上官网yarn参考其他安装方法。
yarn -version或者yarn -V参考yarn版本

使用

项目下初始化

其实跟npm一样, 在项目文件夹下运行命令行,输入yarn init,接下来就跟npm init一样,询问你

  • name : 项目名
  • version : 版本号
  • description : 项目描述
  • entry point : 入口文件
  • git repository： git项目链接地址
  • author : 作者
  • license : 协议

然后会自动生成package.json文件

添加包

通过命令行yarn add [package]安装需要的包,会自动添加到package.json中,还可以通过yarn add [package]@[version]带版本号安装
用yarn upgrade [package]更新包,同样的yarn upgrade [package]@[version]更新到某一版本
yarn add [package] --dev
yarn有一个功能————离线安装包依赖
命令:yarn add [package] --offline

安装依赖

跟npm install类似的命令yarn install,yarn也可以直接用yarn就安装

删除包

yarn remove [package]删除包命令

其他命令

yarn cache ls在本地缓存中的包
yarn cache dir缓存所在路径
yarn cache clean清除本地缓存
yarn config list列出config
yarn config set registry registry.npm.taobao.org换淘宝镜像
yarn info [package]查看包信息,本地或线上都可以
yarn global全局
yarn self-update更新yarn
yarn why [package]分析为什么要安装这个包
yarn run类似npm run
yarn link发布包

yarn管理器有一个很重要的文件需要注意，就是yarn.lock，这个是用来依赖的正确性，快速可靠安装的；是执行cli的时候自动生成的，在项目的根目录下，需要保留！！！！不要编辑它，这是自动生成的
在其他电脑初始化，必须记得把package.json和yarn.lock复制过去
