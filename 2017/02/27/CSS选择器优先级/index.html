CSS选择器优先级

February 27 2017

优先级

选择器分为4个等级：

  • 内联样式具有最高的优先级
  • ID选择器（例如：#example）
  • 类选择器（class selectors） (例如：.example)，属性选择器（attributes selectors）（例如：[type="radio"]），伪类选择器（pseudo-classes）（例如：:hover）
  • 元素(类型)选择器（type selectors）（例如：h1）和 伪元素选择器（pseudo-elements）（例如：:before）

注意：!important 规则例外。当在一个样式声明上使用 !important 规则时，该样式声明会覆盖CSS中任何其他的声明。所以尽量避免使用。

怎样覆盖!important? 用优先级更高的选择器。

优先级的计算（示例）

选择器 特殊性
style="" 1,0,0,0
#wrapper{} 0,1,0,0
.date{} 0,0,1,0
div{} 0,0,0,1
#wrapper #content{} 0,2,0,0
#content .date{} 0,1,1,0
div#content{} 0,1,0,1
p.date{} 0,0,1,1
div p{} 0,0,0,2

“Text”是什么颜色呢？

绿色的，因为第一个优先级（1,0,0,2），第二个优先级（0,0,0,1），第三个优先级（0,0,0,2）,第一个优先级最高
